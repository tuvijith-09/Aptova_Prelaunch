# Aptova

An AI-driven matching system that helps students, fresh graduates, and early earners in India discover income-generating activities tailored to their skills, budget, location, and time. The name comes from *aptus* (Latin: fit, suited) + *nova*.

## Language

**Opportunity**:
Any income-generating activity Aptova can suggest to a user — business ideas, gigs, side hustles, or full ventures. The core unit of output. Aptova's database contains 135+ pre-researched Opportunities across categories like Zero Capital, Part-Time, Tier 3/Rural.
_Avoid_: Job, listing, recommendation

**Assessment**:
The conversational AI session where the user answers ~10 adaptive questions. The AI extracts structured Profile data from natural language responses. No two Assessments are the same. This is the entry point to the product.
_Avoid_: Quiz, survey, form, questionnaire

**Profile**:
The structured data Aptova extracts from the Assessment — location, capital (₹), time availability (hours), time slot, skills, interests, experience. The user never fills a form; the Profile is built from the conversation.
_Avoid_: Account, registration form, application

**Report**:
The deliverable Aptova produces after the Assessment — the user's top 3 matched Opportunities, each with a Fit Score, reasoning, and next steps.
_Avoid_: Results, output, suggestions page

**Fit Score**:
A percentage indicating how well an Opportunity matches a specific user's Profile. Appears in the Report alongside each Opportunity. Three tiers: Strong Fit, Moderate Fit, Weak Fit.
_Avoid_: Rating, rank, match percentage

**Validation**:
A second-stage deep dive where the user picks one Opportunity from their Report and answers ~5 focused follow-up questions. Produces a readiness verdict with strengths, gaps, and concrete next steps. Deferred to post-beta.
_Avoid_: Verification, confirmation

**Explore Catalogue**:
The public browse page showing all 135+ Opportunities organized by category (Zero Capital, Part-Time, Tier 3/Rural, etc.) with filter and sort. Users can browse without an account. Each card links to the Assessment via a "Check my fit" button.
_Avoid_: Database, listings page, marketplace
